# 集成点和外部依赖

## 当前集成

| 服务 | 用途 | 集成类型 | 关键文件 |
|------|------|----------|----------|
| Better Auth | 用户认证 | 内置库 | `apps/server/src/lib/auth.ts` |
| Turso | 数据库 | SQLite 兼容 | `apps/server/drizzle.config.ts` |

## 内部集成点

- **前后端通信**: tRPC over HTTP，端口 3000（服务器）到 3001（客户端）
- **认证流**: Better Auth 通过 `/api/auth/**` 路径处理
- **类型共享**: `AppRouter` 类型从服务器导入到客户端
